# CleanPokedex
- **Clean Architecture, 함수형, 프로토콜 지향 패러다임 이해**를 위한 **토이 프로젝트** 입니다.

### API

- [PokeAPI](https://pokeapi.co/)

### **구현 사항**

- UITableView (List & Detail)
- Kingfisher Image 세팅
- List Pagenation
- infinity Scroll
- Pokemon Detail 불러오기


### **사용 기술**

- Clean Architecture
- UIKit(Code base)
- URLSession
- RxSwift, RxCocoa
- Kingfisher
